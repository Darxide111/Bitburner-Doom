You can't play this right from the start of Bitburner, you'll need singularity access or 64 GB of ram on home.
Download or wget install.js, then run install.js, then run doom.js once it's done.
Only the 3rd room is populated with enemies at the moment. You can add more to map.txt by replacing the dots, use the map key.
Map key:
a = ammo
h = health
P = player start
i = imp
c = cacodemon
Y = cyberdemon
d = demon
z = zombie
l = lost soul
